PineTree API

Create hosted checkout sessions, accept Solana, Base, Lightning, and Shift4 payments, then fulfill from signed webhooks.

https://app.pinetree-payments.com/api/v1

Authentication

Secret API keys

Use `pt_live_*` keys on servers with `Authorization: Bearer`.

Browser public keys

Use `pk_live_*` keys with `X-PineTree-Public-Key` for browser checkout creation only.

Authorization: Bearer pt_live_...
Content-Type: application/json
Idempotency-Key: order_123

Endpoint Reference

POST/api/v1/checkout/sessions

Create a hosted checkout session. Scope: `checkout.sessions:create`.

GET/api/v1/checkout/sessions

List sessions. Scope: `checkout.sessions:read`.

GET/api/v1/checkout/sessions/{id}

Retrieve a session. Scope: `checkout.sessions:read` or `checkout.sessions:create`.

POST/api/v1/checkout/sessions/{id}/cancel

Cancel an open session. Scope: `checkout.sessions:write` or `checkout.sessions:create`.

POST/api/v1/checkout/sessions/{id}/expire

Expire an open session. Scope: `checkout.sessions:write` or `checkout.sessions:create`.

GET/api/v1/payments/{id}

Retrieve a payment. Scope: `payments:read`.

GET/api/v1/webhook-deliveries

List delivery records. Scope: `webhooks:read`.

POST/api/v1/webhook-deliveries/{id}/retry

Retry a delivery. Scope: `webhooks:write`.

POST/api/v1/browser/checkout/sessions

Create checkout from browser code using `X-PineTree-Public-Key`.
